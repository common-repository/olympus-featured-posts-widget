== Description ==

**Featured Posts Widget**

Display your most important posts in the sidebar.

Compatible with all themes.

= Get involved =

Developers can contribute to the source code on the ['Olympus Featured Posts Widget' GitHub Repository](https://github.com/DannyCooper/olympus-featured-posts-widget).

== Installation ==

1. Upload `olympus-featured-posts-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the widget to a sidebar or other widget area.
